Building a Compiler from Scratch

A concept-first guide to compiler construction, with the L1S compiler (Ignis) as the running example.

v2 — Pedagogical Edition

This book teaches compiler construction from first principles. Each chapter introduces a general concept — lexing, parsing, type checking, code generation — then shows how the concept is realized in a real, working compiler written in Rust. The goal is not just to explain what a particular codebase does, but why compilers are designed the way they are, and how you'd build one yourself.

Source → Lexer → Parser → Type Checker → IR Builder → Optimizer → Codegen → nasm + cc → Executable
