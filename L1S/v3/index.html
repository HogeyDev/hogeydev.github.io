Building a Compiler

A hands-on, first-principles journey through compiler construction. Two compilers, one book — Rust and OCaml, side by side.

What is this? This is a textbook that builds two complete compilers from the ground up, one chapter at a time. We don't start with a grand design. We start with a simple question — "how does a computer understand my code?" — and follow the thread wherever it leads. Each chapter solves one concrete problem. The next chapter introduces the problem that the previous solution creates. By the end, we'll have two working compilers (one in Rust, one in OCaml) and a deep understanding of how they work.

Chapters

  1. The SparkWhy compilers? What will we build? The big picture.
  2. Raw MaterialsSource code as text. Positions, spans, reading files.
  3. Finding the WordsBreaking text into tokens. Keywords, identifiers, literals.
  4. Making Sense of StructureFrom flat tokens to nested trees. Recursive descent parsing.
  5. The Shape of a ProgramThe Abstract Syntax Tree. What the parser really builds.
  6. Does This Make Sense?Type checking. Symbol tables. Catching mistakes.
  7. Bridging the GapAn intermediate language between AST and assembly.
  8. Speaking the Machine's LanguageGenerating x86-64 assembly. Registers, stack frames, calling conventions.
  9. The Final MileDriving the pipeline. Error reporting. Assembling and linking.
  10. Where Next?What we built. What comes next. Further exploration.

Companion Code

Every line of code in this book is drawn from two real, working compilers:

  • Rust compiler: code/rust/ — a complete implementation in Rust
  • OCaml compiler: code/ocaml/ — a complete implementation in OCaml

Both compile the same language to x86-64 assembly. Both are under 2000 lines. Both are designed to be read alongside this book.

To build either compiler, see the Makefile or README in each directory.

The Language We'll Build

The language we compile doesn't exist yet. We'll design it together, one decision at a time. Every feature is motivated by a concrete need that arises during our journey. The end result is a small but expressive language — similar in spirit to Ignis, but discovered naturally rather than imposed from above.

About v3

This is the third version of the compiler textbook. v1 was a codebase tour of the Ignis compiler. v2 was a concept-first pedagogical reboot. v3 is a complete rewrite that builds two compilers from first principles, side by side in Rust and OCaml.
