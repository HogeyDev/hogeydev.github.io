Building a Compiler from Scratch

A complete walkthrough of the L1S compiler — a real, working compiler that translates a C-like language into x86-64 machine code, written in Rust.

This book takes you through every stage of a production-quality toy compiler, from reading source code to emitting NASM assembly. Each chapter explains the what, the why, and the how, with annotated code from the actual compiler.

Source → Lexer → Parser → Type Checker → IR Builder → Optimizer → Codegen → nasm + cc → Executable
